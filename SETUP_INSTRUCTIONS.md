# Setup Instructions

## Uploading to GitHub

### Method 1: GitHub Web Interface (Easiest)

1. Go to github.com and sign in
2. Click "New repository" (green button)
3. Name it: `ai-systems-thinking-workshop`
4. Make it Public
5. Don't initialize with README (we have one)
6. Click "Create repository"
7. On the next page, click "uploading an existing file"
8. Drag and drop ALL files from this folder
9. Commit with message: "Week 1: Core workshop materials"

### Method 2: Command Line (If you're comfortable with Git)

```bash
cd path/to/ai-systems-thinking-workshop
git init
git add .
git commit -m "Week 1: Core workshop materials"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/ai-systems-thinking-workshop.git
git push -u origin main
```

## After Upload

1. **Pin the repo** to your GitHub profile (makes it visible to visitors)
2. **Create the 5 GitHub Issues** using templates in GITHUB_ISSUES.md
3. **Update README** - Replace "yourusername" with your actual GitHub username
4. **Update CV** - Add link to this repo under "Projects" or "Training Materials"
5. **Share on LinkedIn** - 3-sentence post linking to the repo
